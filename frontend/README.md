# VitaBalance Frontend

Frontend-ul aplicației VitaBalance - Sistem de recomandare nutrițională personalizată.

## Tehnologii

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion (animații)
- Recharts (grafice)
- Axios (HTTP client)
- jsPDF (export PDF)

## Instalare

```bash
npm install
```

## Rulare

```bash
npm run dev
```

Aplicația va rula pe `http://localhost:3000`

## Build pentru producție

```bash
npm run build
```
